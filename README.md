# FastEncryption
C#에서 여러 암호 알고리즘을 구현한 뒤 유니티 기반 실제 멀티 플레이 게임을 만들고, 암호 알고리즘 적용 및 알고리즘별 성능 비교.
<br><br>

## Block Cipher
### Key: 128 bits
#### AES, ARIA, SPECK
<br>

### Key: 64 bits
#### HIGHT, TWINE
<br>

### Mode of Operation
#### ECB, CBC, CFB, OFB, CTR, GCM
<br>

## Public-Key Cryptography
### RSA
#### Key: 2048 bits, Hash: SHA-256 (표준 구현 X)

### ECC
#### Key: 256 bits, Hash: SHA-256, NIST P-256
